# O que é o OneSignal?

Serviço de Push Notification para sites e aplicativos móveis. Que oferece suporte a todas as principais plataformas, fornecendo SDKs dedicados para cada plataforma, uma API de servidor RESTful e um painel on-line para os profissionais de marketing projetarem e enviarem notificações por push.

---

# Mas por quê?

>Permitir que os desenvolvedores se concentrem no envolvimento do usuário com a qualidade do produto ao invés de uma implementação complexa
<!-- 
  Essa complexidade muitas vezes se dá pelo fato de ser necessário gerenciar
  o token que é gerado no momento em que o dispositivo se comunica com a API de notifications, 
  e que ao utilizar um sistema de Notifications,como Firebase, por exemplo, 
  este token sofre alterações de tempos em tempos, Tornando um pouco complicado 
  identificar o momento em que seu Backend deve receber um novo token.
 -->
